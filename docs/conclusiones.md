# Conclusiones

## Aprendizajes obtenidos

El desarrollo de UndertakeIt ha permitido aplicar de forma práctica los conocimientos adquiridos durante el ciclo formativo de Desarrollo de Aplicaciones Web.

A lo largo del proyecto se han trabajado áreas como el desarrollo backend con PHP, la gestión de bases de datos MySQL, el desarrollo frontend mediante HTML, CSS, JavaScript y Tailwind CSS, así como la organización de aplicaciones web utilizando una estructura inspirada en el patrón MVC.

Además, el proyecto ha servido para adquirir experiencia en la planificación, diseño, implementación y despliegue de una aplicación web completa.

---

## Problemas encontrados durante el desarrollo

Uno de los principales desafíos del proyecto ha sido gestionar el alcance de las funcionalidades previstas inicialmente.

Aunque la base de datos y la estructura general de la aplicación fueron diseñadas pensando en futuras ampliaciones, algunas funcionalidades como el calendario, los hábitos o las subtareas recursivas no pudieron completarse dentro del tiempo disponible.

También fue necesario adaptar parte de la estructura del proyecto para ajustarse a las limitaciones del servicio de alojamiento utilizado durante el despliegue.

---

## Valoración personal

UndertakeIt ha supuesto el proyecto más completo desarrollado durante el ciclo formativo y ha permitido integrar conocimientos de múltiples módulos en una única aplicación.

El resultado final es una plataforma funcional de gestión de tareas con soporte para trabajo colaborativo mediante grupos y roles de usuario, capaz de servir como base para futuras ampliaciones.

A pesar de las funcionalidades pendientes, los objetivos principales planteados al inicio del proyecto han sido alcanzados satisfactoriamente, obteniendo una aplicación estable, organizada y preparada para continuar evolucionando.
