# Introducción

## Descripción del proyecto

UndertakIt, es una aplicación web de gestión de tareas, el nombre es un juego de palabras inglesas “undertaking” (proyecto o tarea) e “It”, como si dijera “hazlo” o “Llévalo a cabo”. 

Esta es una plataforma estilo To-Doo list para una organización personal y colaborativa, donde administrar tareas en base a horarios y calendarios.

UndertakeIt es una aplicación web orientada a la gestión personal y colaborativa de tareas. Su objetivo es proporcionar una herramienta sencilla y organizada que permita a los usuarios planificar actividades, realizar un seguimiento de su progreso y mantener la información relacionada con sus tareas en un único lugar. Además, se conforma de un sistema de grupos que permite la colaboración entre usuarios mediante distintos roles y niveles de permisos, facilitando la administración compartida de tareas y proyectos.

La aplicación permite crear, editar y completar tareas, añadir archivos adjuntos y generar documentos PDF con la información almacenada. Además, la estructura del proyecto ha sido diseñada para permitir futuras ampliaciones, como la incorporación de calendarios, hábitos y sistemas avanzados de subtareas.

## Objetivos del proyecto

Los principales objetivos de UndertakeIt son:

* Desarrollar una aplicación web completa utilizando tecnologías estudiadas durante el ciclo formativo de Desarrollo de Aplicaciones Web.
* Aplicar una arquitectura organizada basada en la separación de responsabilidades entre modelos, vistas y controladores.
* Diseñar una base de datos capaz de soportar la gestión de usuarios, tareas, archivos y relaciones entre entidades.
* Implementar funcionalidades reales de gestión de tareas mediante una interfaz intuitiva.
* Adquirir experiencia en el desarrollo de proyectos web de tamaño medio, incluyendo diseño, programación, bases de datos y despliegue.

## Objetivos de aprendizaje

Durante el desarrollo del proyecto se han puesto en práctica conocimientos relacionados con:

* Programación en PHP.
* Desarrollo de interfaces web con HTML, CSS y Tailwind CSS.
* Programación cliente con JavaScript y jQuery.
* Diseño y gestión de bases de datos MySQL.
* Gestión de sesiones y autenticación de usuarios.
* Organización de proyectos mediante una arquitectura estructurada.
* Generación de documentos PDF mediante DomPDF.
* Despliegue de aplicaciones web en un entorno de producción.
