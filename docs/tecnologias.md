# Tecnologías utilizadas

## Backend

### PHP

PHP ha sido el lenguaje principal utilizado para el desarrollo de la lógica de negocio de la aplicación. Se emplea para gestionar las peticiones del usuario, la autenticación, el acceso a la base de datos y la generación dinámica de contenido.

### MySQL

MySQL se utiliza como sistema gestor de bases de datos relacional. Permite almacenar la información de usuarios, tareas, grupos, archivos adjuntos y demás entidades necesarias para el funcionamiento de la aplicación.

---

## Frontend

### HTML5

HTML5 se emplea para estructurar el contenido de las distintas páginas de la aplicación.

### Tailwind CSS

Tailwind CSS es el framework CSS utilizado para el diseño de la interfaz.

### JavaScript

JavaScript se utiliza para añadir interactividad a la aplicación, gestionar eventos del usuario y mejorar la experiencia de uso.

### jQuery

jQuery complementa el uso de JavaScript simplificando la manipulación del DOM y la gestión de eventos en determinadas funcionalidades.

---

## Librerías y herramientas externas

### DomPDF

DomPDF permite generar documentos PDF a partir de plantillas HTML, facilitando la exportación de información desde la aplicación.

### Font Awesome

Font Awesome proporciona la colección de iconos utilizada en la interfaz de usuario.

### Fancybox

Fancybox se utiliza para la visualización ampliada de imágenes y archivos adjuntos dentro de la aplicación.

### Slick Carousel

Slick Carousel se utiliza para implementar carruseles de imágenes en la vista de *showcase*.

---

## Entorno de desarrollo

### Composer

Composer se ha utilizado como gestor de dependencias para instalar y mantener la librería de [DomPDF](#DomPDF).

### Git y GitHub

Git se ha empleado para el control de versiones del proyecto, mientras que GitHub se ha utilizado para el alojamiento del repositorio y la documentación mediante GitHub Pages.
